¿Cómo usar la API?

Usa tu token de sesión (cookie) en futuras solicitudes a los endpoints protegidos. Por ejemplo, para subir un archivo: 

          
            fetch('/api/files/upload', {
              method: 'POST',
              body: formData, // Asegúrate de incluir el archivo aquí
              // Tus cookies de sesión serán enviadas automáticamente por el navegador
            });

Para descargar, puedes visitar:

          
            /api/files/:jobId/download
            /api/files/:jobId/errors

Para acceso programático (desde otra aplicación), considera implementar JWTs u otro mecanismo si no quieres depender de cookies de sesión.
