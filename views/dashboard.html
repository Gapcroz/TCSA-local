¿Cómo usar la API?

Si obtuviste un JWT, úsalo en el encabezado `Authorization: Bearer <token>` para todas las solicitudes a la API. 

          
            fetch('/api/files/upload', {
              method: 'POST',
              headers: {
                'Authorization': `Bearer TU_JWT_AQUI`
              },
              body: formData
            });

Para descargar, puedes visitar:

          
            /api/files/:jobId/download
            /api/files/:jobId/errors
